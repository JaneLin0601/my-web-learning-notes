1️⃣ transform — 变换

transform 让元素可以 旋转、缩放、位移、倾斜，不改变文档流布局。

rotate(deg) 旋转

围绕中心旋转
transform: rotate(45deg);

scale(n) 缩放

放大/缩小
transform: scale(1.2);

translate(x, y) 位移

移动位置
transform: translate(20px, 10px);

skew(deg) 倾斜

拉伸倾斜
transform: skew(10deg);
悬停我

rotate + scale + translate

/* 2D 变换组合 */
.box:hover {
  transform: rotate(45deg) scale(1.2) translateX(20px);
}

2️⃣ transition — 过渡

transition 让属性变化时 平滑过渡，而不是瞬间跳变。

transition-property

要过渡的属性
all / background / transform

transition-duration

过渡时长
0.5s / 1s / 300ms

transition-timing-function

速度曲线
ease / linear / ease-in / ease-out

transition-delay

延迟时间
0s / 0.2s
悬停我

颜色 + 圆角 + 缩放

/* 简写：property duration timing delay */
.box {
  transition: all 0.5s ease 0s;
}
.box:hover {
  transform: scale(1.1);
}

3️⃣ animation + @keyframes — 动画

@keyframes 定义动画关键帧，animation 调用并控制播放。

animation-name

动画名称
heart / slide / pulse

animation-duration

播放时长
3s / 1.5s

animation-timing-function

速度曲线
ease / linear / ease-in-out

animation-iteration-count

播放次数
infinite / 2 / 3

animation-direction

播放方向
normal / reverse / alternate

animation-play-state

播放状态
running / paused
动起来

自动呼吸 + 变色 + 变圆

/* 定义动画 */
@keyframes pulse {
  0% { transform: scale(1); }
  50% { transform: scale(1.2); }
  100% { transform: scale(1); }
}

/* 调用动画 */
.box {
  animation: pulse 2s ease-in-out 0s infinite;
}

4️⃣ 悬停暂停 — animation-play-state: paused

鼠标悬停时 暂停动画，移出后继续播放。

.box:hover {
  animation-play-state: paused;
}
悬停暂停

悬停 → 动画暂停

5️⃣ clip-path — 星形裁剪

clip-path 用 polygon() 坐标点裁剪出 星形， 配合动画可做旋转呼吸星。

⭐ 星形 · 悬停暂停

/* 星形 clip-path：10 个坐标点 */
.star {
  clip-path: polygon(
    50% 0%, 61% 35%, 98% 35%,
    68% 57%, 79% 91%, 50% 70%,
    21% 91%, 32% 57%, 2% 35%, 39% 35%
  );
}

/* 星形旋转动画 */
@keyframes starSpin {
  0% { transform: rotate(0deg); }
  100% { transform: rotate(360deg); }
}

💡 口诀：10 个点画星形，5 个外凸 + 5 个内凹，交替排列。

6️⃣ 3D 变换 进阶

在 2D 变换基础上增加 Z 轴，实现立体旋转效果。

rotateX(deg)

绕 X 轴旋转
transform: rotateX(45deg);

rotateY(deg)

绕 Y 轴旋转
transform: rotateY(45deg);

perspective

透视深度（父级设置）
perspective: 500px;

transform-style

3D 空间保留
transform-style: preserve-3d;
3D

悬停 → 立体旋转

/* 3D 旋转 */
.box-3d:hover {
  transform: rotateX(30deg) rotateY(40deg) scale(1.05);
  box-shadow: 0 20px 60px rgba(0,0,0,0.4);
}

7️⃣ 对比总结

属性 作用 触发方式
transform 旋转/缩放/位移/倾斜 直接改变
transition 属性变化平滑过渡 悬停 / 点击 / JS
animation 复杂关键帧动画 页面加载自动播放
clip-path 裁剪出任意形状 配合 transform 做形状动画

8️⃣ 记忆口诀 🧠

transition 过渡做衔接，@keyframes 动画定义，
animation 调用无限次，clip-path 剪出星星来。”
