⚡ VSCode 必记快捷键

🚀 神键（高频区）

  • Ctrl + Shift + P：命令面板（万能入口）
  • Ctrl + P：快速打开文件
  • Ctrl + F：当前文件查找
  • Ctrl + H：当前文件替换
  • Ctrl + /：注释/取消注释
    • Ctrl+k+c
  • Alt + ↑/↓：整行上下移动
  • Ctrl + D：多选相同词
  • Ctrl + Shift + F：全局搜索
  • Ctrl + ~：打开/关闭终端
  • Ctrl + S：保存（救命键）

🛠️ 基础操作

  • Ctrl + C / V / X：复制/粘贴/剪切
  • Ctrl + Z / Y：撤销/重做
  • Ctrl + W：关闭当前编辑的文件
  • Ctrl + Tab：在打开的文件间切换

🎨 界面控制

  • Ctrl + B：显示/隐藏左侧资源管理器
  • Ctrl + J：显示/隐藏底部面板
  • F11：全屏模式

✨ 格式化与复制

  • Shift + Alt + F：格式化代码 ⭐
  • Shift + Alt + ↓：复制当前行到下一行
  • Shift + Alt + ↑：复制当前行到上一行
  • Ctrl + Shift + K：删除当前行
