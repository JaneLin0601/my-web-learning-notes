✨ CSS3 新特性 · 核心要点

CSS3 让网页从「平面」走向「立体」

1️⃣ CSS3 是什么

CSS3 是 CSS 的 最新版本，它把 CSS 从单纯的“样式表”变成了 可以做动画、渐变、变形、滤镜 的“设计语言”。

💡 核心思想：让网页更生动、更精致，减少对图片的依赖

2️⃣ 最常用的 CSS3 新特性

border-radius 圆角

给元素加圆角，告别直角
border-radius: 8px; → 50% 变圆形

box-shadow 阴影

给元素加立体阴影，更有层次
box-shadow: 0 4px 20px rgba(0,0,0,0.3);

linear-gradient 渐变

颜色平滑过渡，比图片更轻量
background: linear-gradient(135deg, #1f6feb, #8b5cf6);

transform 变换

旋转、缩放、倾斜、位移
transform: rotate(15deg) scale(1.2);

transition 过渡

属性变化时平滑动画，交互更自然
transition: all 0.5s ease;

animation 动画

关键帧动画，更复杂的动态效果
animation: pulse 2s ease-in-out infinite;

filter 滤镜

模糊、亮度、对比度等图像特效
filter: blur(4px) brightness(1.2);

flexbox 弹性盒子

一维布局，告别浮动
display: flex; justify-content: center;

4️⃣ 常用代码片段 📋

圆角 + 阴影（卡片）

.card {
  border-radius: 12px;
  box-shadow: 0 8px 30px rgba(0,0,0,0.3);
}

渐变背景

.hero {
  background: linear-gradient(135deg, #1f6feb, #8b5cf6);
}

悬停放大 + 旋转

.btn:hover {
  transform: scale(1.1) rotate(3deg);
  transition: 0.4s ease;
}

毛玻璃效果

.glass {
  background: rgba(255,255,255,0.08);
  backdrop-filter: blur(14px);
}

5️⃣ 浏览器兼容性 ⚠️

✅ 完全支持

border-radius, box-shadow, gradient, transform, flexbox

⚠️ 需加前缀

backdrop-filter（-webkit-）

❌ IE 不支持

所有 CSS3 新特性在 IE 中均不支持

💡 现代浏览器（Chrome / Edge / Firefox / Safari）都完全支持 CSS3，放心使用。
