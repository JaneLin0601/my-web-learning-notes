💻 Windows 快捷键 (前端开发版)

高频系统控制
- Win + I：打开设置
- Win + E：打开“此电脑”
- Win + D：显示桌面
- Win + L：锁定电脑
- Win + V：打开剪贴板历史 ⭐
- Win + S：打开搜索
- Win + R：打开“运行”窗口

窗口管理与虚拟桌面
- Win + 左/右：窗口分屏
- Win + Tab：任务视图
- Win + Ctrl + D：新建虚拟桌面

开发专用
- Win + .：打开Emoji面板 😊
- Win + Shift + S：截图工具
- Win + G：屏幕录制
- Alt + Tab：切换应用
